# Limitations

- Download counts measure package downloads, not unique users or active use.
- Repository files are signals of practice, not proof that a process is effective.
- Static-analysis findings are tool-dependent observations, not defects by definition.
- Python metrics do not generalise automatically to other languages.
- CLI probes cannot establish scientific correctness.
- Upstream default-branch state may differ from the packaged release and is analysed separately.
- Failed audit infrastructure yields `ERROR`, never an inferred package failure.
- Agent review is bounded and sampled; confidence and adjudication are recorded.
